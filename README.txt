This assignment is been written in SCALA and the test cases are written in Java with JUnit4 .

Dependencies
SCALA version 2.10.2
JRE 1.7
Jnit 4 

Artifacts
Model classes are in package model inside the src directory
Test cases are in package model inside the test directory

Running application
This is the eclipse project, please import the project into eclipse to run the test cases 
or you can run it form the command prompt as :-
java -cp .:/usr/share/java/junit.jar org.junit.runner.JUnitCore [test class name]
